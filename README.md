# Jobs Board Platform

A full-stack job board platform built with the **MERN** stack: MongoDB, Express, React, and Node.js.  
Employers can post jobs and job-seekers can browse and apply. The platform is responsive and designed for real-world usage.

---

## Features

- User roles: **Employer** and **Job Seeker**  
- Employers can create, edit, and delete job posts  
- Job Seekers can view job listings, filter/search, and apply for jobs  
- User authentication and authorization  
- RESTful API backend using Node.js & Express  
- MongoDB database with Mongoose  
- React frontend with modern UI and routing  
- Responsive design for mobile and desktop  

---

## Technologies

- **Backend:** Node.js, Express.js  
- **Frontend:** React.js, React Router, Axios  
- **Database:** MongoDB (local or Atlas)  
- **Authentication:** JWT tokens  
- **Styling:** CSS / Sass / Styled Components
